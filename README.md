# factors classifier (C++)

## Description
This C++ program takes an integer limit from the user and outputs each number from 1 up to that limit.  
and checks if each number is:
- A factor of 3
- A factor of 5
- A factor of both 3 and 5
- None 

The program is done using multiple files to demonstrate C++ code structure.
